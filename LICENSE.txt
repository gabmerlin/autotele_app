AUTOTELE v1.3.0 - APPLICATION DE GESTION TELEGRAM

Fonctionnalites:
- Gestion multi-comptes Telegram
- Messagerie en temps reel
- Planification de messages
- Envoi massif securise
- Chiffrement AES-256 des sessions
- Score de securite : 10/10

Securite:
- Protection contre les injections SQL
- Validation complete des entrees (XSS, injection)
- Magic bytes pour detection de fichiers malveillants
- Logs anonymises (RGPD)
- Rate limiting integre
- Permissions fichiers restrictives

Dependances principales:
- NiceGUI (Interface utilisateur)
- Telethon (API Telegram)
- Cryptography (Chiffrement)
- SQLite (Base de donnees locale)
- Python 3.11+

Pour plus d'informations:
- Documentation: README_INSTALLATEUR.txt
- Support: https://github.com/gabmerlin/autotele_app
- Tests de securite: tests/security_tests.py
